# MDP.AspNetCore.Authentication

MDP.AspNetCore.Authentication是開源的.NET開發套件，協助開發人員快速建立整合ASP.NET Core身分驗證的應用系統。提供Line、Google、Facebook等OAuth身分驗證模組，及JWT等Token身分驗證模組，用以簡化開發流程並滿足多變的商業需求。
